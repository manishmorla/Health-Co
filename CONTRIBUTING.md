# Contributing to Health & Co

Thank you for your interest in contributing to Health & Co! This document provides guidelines and instructions for contributing.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/yourusername/Health-Co.git`
3. Create a virtual environment and install dependencies:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   pip install -r requirements.txt
   ```
4. Create a `.env` file from `.env.example` and add your API key
5. Create a new branch: `git checkout -b feature/your-feature-name`

## Code Style

- Follow PEP 8 style guidelines
- Use meaningful variable and function names
- Add docstrings to functions and classes
- Keep functions focused and single-purpose

## Submitting Changes

1. Make your changes
2. Test your changes thoroughly
3. Commit your changes with clear messages
4. Push to your fork
5. Open a Pull Request with a clear description

## Areas for Contribution

- UI/UX improvements
- Additional nutrition analysis features
- Better error handling
- Documentation improvements
- Performance optimizations
- Testing

## Questions?

Feel free to open an issue for any questions or suggestions!
